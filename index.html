¿Que es el Delegado de Redes?

El Delegado de Redes será la persona del Curso que se encargue de manejar Grupos y Cuenta de nuestras redes compartidas como por Ej: Tiktok, Instagram, ETC.

¿Qué necesito para Postularme?

Necesitas Saber estas cosas:

  • - Ser responsable con Las Redes
  • - Saber Manejarlas
  • - Comprometerse a Estar al tanto de las Redes
  • - Informar a los Demas de X Suceso
  • - Hacer las Tareas Necesarias para Nuestras Redes

¿Alguna recomendación para guiarme?

Se les recomienda a los postulados Saber por lo menos Guiarse con las redes, También tendrán el derecho de elegir Camarografos y editores para ayudarlo. También recomendamos Una actitud Alegre y Educada en casos importantes, Ej:

  • Empatia, Ser abierto, Realista, responsable, Y sin vergüenza para cualquier suceso que se necesite grabar o hablar en publico.
